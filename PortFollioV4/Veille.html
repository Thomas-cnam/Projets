Introduction :


1. Vue.js


Vue.js c'est un framework JavaScript populaire pour la création d'interfaces utilisateur dynamiques et réactives

Vue.js est un framework JavaScript open source créé par Evan You en 2014.

Il est conçu pour faciliter la construction d'interfaces web interactives en adoptant une approche basée sur les composants.

Vue.js se concentre sur la simplicité et la flexibilité, ce qui le rend adapté aux petits projets et aux applications web plus complexes.

2. Les fonctionnalités de Vue.js

Composants : Vue.js permet de créer des composants réutilisables, facilitant ainsi la gestion de l'interface utilisateur

Les directives de Vue.js offrent un moyen déclaratif de manipuler le DOM, permettant ainsi un contrôle précis sur la façon dont les éléments sont rendus.

Réactivité : Le système de réactivité de Vue.js met automatiquement à jour les éléments affectés par les changements de données.

L'implémentation

Il faut inclure la bibliothèque Vue.js dans notre page web grâce à une balise script.

La création d'une balise racine qui permettra le fonctionnement de Vue.js.

Ainsi que la création d'une nouvelle vue avec son instance

Cette vue permet donc de définir les données de l'application, ainsi que la liaison entre la balise racine et l'instance

Différents test

Vue 1

L'affichage d'un nom et d'un message qui sont défini dans l'instance de la vue



Vue 2

La création d'un champs input pour pouvoir rentrer le nom de l'user et que cela change le message qui afficher le nom de l'user préalablement défini.

Avec l'utilisation de la directive v-model="user" qui permet donc de créer une liaison entre un champ de formulaire et une propriété de l'instance Vue.

Ainsi que la création d'un tableau avec différents produits dans l'instance : Utilisation de la dirrective v-for qui permet d'itérer sur une liste ou un tableau et de générer dynamiquement des éléments.



Vue 3

L'ajout d'un bouton commander qui nous permet de récupérer le nom du produit selectionné avec la directive v-on:click qui permet d'attacher des méthodes à leur déclenchement, ici l'utilisation du onclick.



Vue 4

La création d'un tableau des commandes pour stockés les éléments commandé dedans.

Ainsi la création d'un champs liste pour pouvoir les affichés en utilisant v-for pour parcourir le tableau produits et les affichés à la suite.

Ce champs n'apparait uniquement si le tableau des commandes possède un élément ou plus. Ici l'utilisation de la directive v-if qui permet de donner une condition d'affichage à la liste.

Ici la création d'un composant fiche produit qui permet d'afficher tous les produits disponibles dans la liste.
